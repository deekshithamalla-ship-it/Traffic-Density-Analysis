# Traffic Density Analysis

This project analyzes traffic density data for 7 days (24 hours each).

## Features
- Data input and storage using CSV
- Daily traffic analysis
- Hourly average calculation
- Peak hour detection
- Ranking days based on traffic
- Graph visualization using matplotlib

## Technologies Used
- Python
- NumPy
- Matplotlib
- CSV

## How to Run
Run the Python file and enter 24 values for each day.
